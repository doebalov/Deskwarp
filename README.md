**Deskwarp** — это легковесное и высокопроизводительное приложение для Windows, которое возвращает легендарный эффект «желейных окон» (Wobbly Windows), знакомый многим по менеджеру окон Compiz из мира Linux. Программа плавно деформирует окна при их перемещении, создавая реалистичную физическую анимацию прямо на вашем рабочем столе Windows 10 или 11.

## ✨ Особенности

- **Аппаратное ускорение:** Рендеринг анимации выполняется с помощью низкоуровневых графических API **Direct3D 11** и **DirectComposition**, что минимизирует задержки и нагрузку на центральный процессор.
- **Реалистичная физика:** В основе деформации лежит классическая физическая модель «масса-пружина» (`SoftBody`), которая рассчитывает силы трения, жесткости пружин и инерцию окон в реальном времени.
- **Уровни реализма (Realism Levels):** Поддержка нескольких встроенных пресетов физики (от легкого и упругого покачивания до сильной и плавной инерции).
- **Идеальная интеграция с ОС:**
  - Полная поддержка системного масштабирования (**High-DPI** и многомониторных конфигураций).
  - Сохранение закругленных углов окон в Windows 11 и поддержка размытых теней (Drop Shadows).
- **Работа в фоне:** Удобное управление через иконку в системном трее (System Tray), сворачивание в трей и автоматический запуск в фоновом режиме.
- **Один экземпляр (Single Instance):** Защита от повторного запуска процесса с использованием локальных сокетов (`QLocalServer`), позволяющая управлять уже запущенным приложением.

## ⚙️ Как это работает

1. **Захват окна:** Специальный компонент `ScreenGrabber` в фоновом потоке перехватывает графическое содержимое перемещаемого или активного окна с помощью системных вызовов Desktop Window Manager (DWM).
2. **Симуляция ткани:** Окно разбивается на сетку узлов (`GridNode`), связанных пружинами (`GridSpring`). При движении мыши или изменении положения окна к узлам прикладывается сила, заставляющая сетку гнуться и колебаться.
3. **Композиция:** Текстура захваченного окна натягивается на деформированную 3D-сетку в шейдерах Direct3D 11 и выводится поверх рабочего стола через DirectComposition.

## 🛠️ Сборка проекта

Проект разработан на C++ с использованием фреймворка **Qt** и нативных API Windows.

### Требования
- Компилятор **MSVC** (с поддержкой C++17 или выше).
- Установленный **Qt 5 / Qt 6** (модули `Core`, `Gui`, `Widgets`, `Network`, `Svg`).
- Windows SDK (для работы с Direct3D 11 и DirectComposition).

### Зависимости линковщика
При сборке проект автоматически подключает следующие системные библиотеки:
`user32.lib`, `gdi32.lib`, `dwmapi.lib`, `d3d11.lib`, `dxgi.lib`, `dcomp.lib`, `d3dcompiler.lib`, `ole32.lib`, `shell32.lib`.

### Сборка через Qt Creator / CMake
1. Откройте файл проекта (`CMakeLists.txt` или `.pro`) в Qt Creator.
2. Выберите комплект сборки (MSVC 64-bit).
3. Скомпилируйте проект в режиме *Release*.

## 📂 Использование

После запуска приложение сворачивается в системный трей. Вы можете:
- Перетаскивать любые стандартные окна Windows, чтобы увидеть эффект в действии.
- Кликнуть правой кнопкой мыши по иконке в трее, чтобы настроить параметры физики, временно отключить эффект или закрыть приложение.

### Аргументы командной строки
- `--background` — запуск приложения сразу в свернутом режиме в системный трей (без вывода главного окна настроек).
